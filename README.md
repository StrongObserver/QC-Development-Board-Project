# QC-Development-Board-Project

Qualcomm development-board experiments and supporting tools.

- `RB5VisionLab/`: Android application for the RB5 Gen2 camera and imaging pipeline.
- `RB5_SR_lab/`: host-side Real-ESRGAN/TFLite super-resolution validation scripts, assets, and results.

See `push_readme.txt` for the repository commit and push convention.
